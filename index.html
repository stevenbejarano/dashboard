Settings

Google Calendar

Create a project at console.cloud.google.com, enable the Calendar API, and create an OAuth 2.0 Web Client ID.

Data Sources — Google Sheets

Open your sheet in Google Sheets and copy the ID from the URL:
docs.google.com/spreadsheets/d/[SHEET-ID]/edit

Performance — Current Week Column

Enter the column letter from your performance sheet that contains this week's data (e.g. I, J, K…). Update this each week. The previous column is used automatically for trend arrows.

Performance Metrics

Each metric reads a single cell from your sheet at the current week column and the row you specify.

WBR Commentary

These values populate the auto-generated WBR draft. Update quarterly numbers each quarter.

SDO Column Mapping (Advanced)

0-based column index in your SDO Log sheet (column A = 0, B = 1, C = 2…). Only change these if your SDO sheet columns differ from the defaults.
